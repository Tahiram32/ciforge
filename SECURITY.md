# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

Because `ciforge` scans for leaked secrets and runs in CI/CD pipelines, we take security very seriously.

If you discover a vulnerability (e.g., a bypass for the secret scanner, or an injection vector in the CLI), **please do not open a public issue.**

Instead, please report it privately:
1. Go to the [Security Advisories](https://github.com/Tahiram32/ciforge/security/advisories) tab in this repository.
2. Click **Report a vulnerability**.

We will try to respond within 48 hours and release a patch as quickly as possible.
